# `calculations`







## class `Calculations`
A simple calculator class for basic arithmetic. 

This class can perform addition and subtraction. 






### `Calculations.add`

```python
add(a: float, b: float) → float
```

Adds two numbers together. 



**Args:**
 
 - <b>`a`</b> (float):  The first number. 
 - <b>`b`</b> (float):  The second number. 



**Returns:**
 
 - <b>`float`</b>:  The sum of the two numbers. 



### `Calculations.subtract`

```python
subtract(a: float, b: float) → float
```

Subtracts the second number from the first. 



**Args:**
 
 - <b>`a`</b> (float):  The number to subtract from. 
 - <b>`b`</b> (float):  The number to subtract. 



**Returns:**
 
 - <b>`float`</b>:  The result of the subtraction.
