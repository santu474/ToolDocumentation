# `calculator`







## `add`

```python
add(a: float, b: float) → float
```

Adds two numbers which are provided as arguments. 



**Args:**
 
 - <b>`a`</b> (float):  The first number. 
 - <b>`b`</b> (float):  The second number. 



**Returns:**
 
 - <b>`float`</b>:  The sum of the two numbers. 




## `subtract`

```python
subtract(a: float, b: float) → float
```

Subtracts the second number from the first number. 



**Args:**
 
 - <b>`a`</b> (float):  The first number. 
 - <b>`b`</b> (float):  The second number. 



**Returns:**
 
 - <b>`float`</b>:  The result of the subtraction. 




## `multiply`

```python
multiply(a: float, b: float) → float
```

Multiplies two numbers. 



**Args:**
 
 - <b>`a`</b> (float):  The first number. 
 - <b>`b`</b> (float):  The second number. 



**Returns:**
 
 - <b>`float`</b>:  The product of the two numbers. 




## `divide`

```python
divide(a: float, b: float) → float | str
```

Divides the first number by the second number. 



**Args:**
 
 - <b>`a`</b> (float):  The numerator. 
 - <b>`b`</b> (float):  The denominator. 



**Returns:**
 
 - <b>`float | str`</b>:  The result of the division, or  an error message if division by zero is attempted.
